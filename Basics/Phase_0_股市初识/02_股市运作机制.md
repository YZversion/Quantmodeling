# 股市运作机制

## 一张图看懂股市

```
   ┌──────────┐     买入/卖出指令      ┌──────────────┐
   │   你     │ ──────────────────→    │  证券交易所   │
   │  (散户)  │                        │  (上交所/深交所)│
   └──────────┘                        └──────────────┘
         │                                   │
         │ 通过                               │ 撮合买卖单
         ↓                                   ↓
   ┌──────────┐                        ┌──────────────┐
   │  券商    │ ←───── 交易通道 ──────→ │   上市公司   │
   │ (中介)   │                        │  (发行股票)   │
   └──────────┘                        └──────────────┘
```

**三个核心角色**：

1. **上市公司**：发行股票、定期披露信息、经营业务
2. **证券交易所**：提供交易场所、撮合买卖、制定规则
3. **券商（证券公司）**：你的中介，帮你下单、保管股票、提供研究

---

## 中国的证券交易所

| 名称 | 成立时间 | 主要板块 | 特点 |
|------|---------|---------|------|
| 上海证券交易所（上交所） | 1990 | 主板、科创板 | 大型国企、科技企业 |
| 深圳证券交易所（深交所） | 1991 | 主板、创业板 | 中小企业、成长型企业 |
| 北京证券交易所（北交所） | 2021 | — | 小型企业、专精特新 |

### 板块说明

- **主板**：成熟期大公司（如贵州茅台、工商银行）
- **科创板**（上交所）：硬科技、高研发投入公司（如中芯国际）
- **创业板**（深交所）：成长型公司（如宁德时代）
- **北交所**：创新型中小企业

## 交易流程：你按下"买入"后发生了什么？

```
1. 你在交易软件输入：买入 贵州茅台 100股 价格 1800元
                ↓
2. 指令通过券商 → 传到证券交易所
                ↓
3. 交易所撮合系统寻找：有没有人愿意以 ≤1800元 卖出茅台？
                ↓
4. 找到了！卖方挂了 1798元 的卖单
                ↓
5. 成功成交！成交价 1798元，你获得 100股茅台
                ↓
6. 券商从你的账户扣款 179,800元 + 手续费
                ↓
7. 100股茅台记入你的证券账户
```

### 集合竞价 vs 连续竞价

**上午9:15-9:25：集合竞价**
- 所有买卖单集中在一起
- 交易所计算一个让成交量最大的价格
- 这个价格就是开盘价

**上午9:30-11:30、下午13:00-14:57：连续竞价**
- 逐笔撮合，来一单就匹配一单
- 这就是盘中不断跳动的价格

**下午14:57-15:00：收盘集合竞价**
- 确定收盘价

## 股市的参与者：谁在和谁交易？

| 参与者 | 资金量 | 特点 | 你面对的对手 |
|--------|--------|------|-------------|
| 散户 | 小 | 信息少、情绪化 | 你自己 |
| 公募基金 | 大（百亿级） | 有研究团队、纪律性强 | 很强 |
| 私募基金 | 中到大 | 灵活、追求绝对收益 | 很强 |
| 保险/社保 | 巨大 | 长期持有、偏好蓝筹 | 巨强 |
| QFII（外资） | 大 | 研究深、偏好龙头 | 巨强 |
| 券商自营 | 中 | 有信息优势 | 强 |

**关键认知**：你作为散户，面对的对手是专业机构。他们有：
- 全职研究团队（几十人分析一个行业）
- 更快的信息获取渠道
- 程序化交易系统
- 百亿级别的资金量

> ⚠️ 这不是吓你，是让你清醒。散户的优势是：灵活性、没有业绩考核压力、可以长期持有。用好这些优势。

## 股市怎么赚钱？（系统层面）

股市的收益来源，从系统角度看：

### 1. 企业成长
公司从小变大，利润从少到多，股价跟着涨。这是股市**最根本的收益来源**。

例子：茅台 2001 年上市时年利润约 3 亿，2024 年年利润超 700 亿。股价涨了近百倍。

### 2. 分红
公司把利润分给股东。长期看，分红复投的回报非常可观。

### 3. 估值波动
市场情绪导致股价偏离"内在价值"。你在低估时买入、高估时卖出，赚估值回归的钱。

> ⚠️ 这是最难赚的钱——因为你要判断"什么是低估/高估"，而市场可能长时间不回归。

## IPO（首次公开发行）

公司第一次把股票卖给公众投资者，就是 IPO。

流程：
1. 公司决定上市 →聘请券商做保荐人
2. 提交申请材料→交易所/证监会审核
3. 审核通过→确定发行价和发行数量
4. 公众投资者申购→资金冻结
5. 中签公布→股票上市交易

**新股申购（打新）**：散户可以参与IPO申购，如果中签，就能以发行价买到新股。新股上市首日通常涨幅较大，但中签率很低（万分之一级别）。
