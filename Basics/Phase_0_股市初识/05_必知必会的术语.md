# 必知必会的术语

> 50 个最常见的股市术语，按类别整理。看完这篇，你就能读懂大部分股市讨论了。

---

## 一、基础概念

| 术语 | 解释 | 举例 |
|------|------|------|
| **A股** | 在中国境内（上交所/深交所/北交所）上市、以人民币计价的股票 | 贵州茅台是A股 |
| **H股** | 在香港上市的内地公司股票 | 腾讯控股是H股 |
| **IPO** | Initial Public Offering，首次公开发行，公司第一次向公众卖股票 | 新股上市 |
| **流通股** | 可以在市场上自由买卖的股票数量 | 区别于限售股（暂时不能卖） |
| **总股本** | 公司所有股票的总数量 | 流通股 + 限售股 |
| **市值** | 股票总数 × 当前股价 = 公司的市场总价值 | 茅台市值 ≈ 2万亿 |
| **流通市值** | 流通股数 × 当前股价 | 真实可交易的规模 |

## 二、交易术语

| 术语 | 解释 | 要点 |
|------|------|------|
| **开盘价** | 当天第一笔成交的价格 | 由9:15-9:25集合竞价决定 |
| **收盘价** | 当天最后一笔成交的价格 | 由14:57-15:00集合竞价决定 |
| **最高价** | 当天成交的最高价格 | |
| **最低价** | 当天成交的最低价格 | |
| **涨停** | 当天股价上涨达到上限，不能再涨了 | A股主板：±10%，科创板/创业板：±20% |
| **跌停** | 当天股价下跌达到下限，不能再跌了 | 同上 |
| **一手** | 100股为最小交易单位 | 必须买100、200、300...不能买150 |
| **T+1** | 今天买入的股票，明天才能卖出 | 不能当天买当天卖 |
| **打新** | 参加新股IPO申购 | 中签率很低，但首日通常涨 |
| **停牌** | 股票暂停交易 | 可能因重大事项、异常波动等 |

## 三、价格与涨跌

| 术语 | 解释 | 举例 |
|------|------|------|
| **涨跌幅** | 今天价格相比昨收的变化百分比 | +2.35% |
| **昨收** | 昨天的收盘价 | 涨跌幅的计算基准 |
| **振幅** | 当天最高价与最低价之间的波动幅度 | (最高-最低)/昨收 × 100% |
| **换手率** | 当天成交量 ÷ 流通股数 × 100% | 换手率高=交易活跃 |
| **量比** | 当天成交量与过去5天平均成交量的比值 | >1说明比平时活跃 |
| **委比** | (买盘量-卖盘量)/(买盘量+卖盘量) × 100% | 正数=买方力量强 |

## 四、估值术语（Phase 2 会深入讲）

| 术语 | 解释 | 初步理解 |
|------|------|---------|
| **PE（市盈率）** | 股价 ÷ 每股利润 | 20倍PE = 你花20元买1元利润，贵不贵？ |
| **PB（市净率）** | 股价 ÷ 每股净资产 | <1说明股价低于公司账面价值 |
| **PS（市销率）** | 股价 ÷ 每股营收 | 适合还没赚钱的成长公司 |
| **ROE** | 净利润 ÷ 股东权益 × 100% | 公司用股东的钱赚了多少回报 |
| **股息率** | 每股分红 ÷ 股价 × 100% | 和银行利息类比，5%股息率=不错的"利息" |
| **EPS** | 每股收益（Earnings Per Share） | 公司利润 ÷ 总股本 |

## 五、市场术语

| 术语 | 解释 | 简单理解 |
|------|------|---------|
| **牛市** | 市场整体上涨的时期 | 大家都在赚钱 |
| **熊市** | 市场整体下跌的时期 | 大多数人在亏钱 |
| **震荡** | 市场在一定区间内上下波动 | 不涨不跌，来回折腾 |
| **大盘** | 通常指上证指数或沪深300 | 代表整个市场的走势 |
| **蓝筹股** | 大型、稳定、盈利好的公司股票 | 如茅台、工商银行 |
| **白马股** | 业绩好、信息透明、被市场认可的好公司 | 和蓝筹类似但更强调增长 |
| **黑马股** | 之前不被关注但突然大涨的股票 | 意外惊喜 |
| **题材股/概念股** | 因某个热点题材（如AI、新能源）被炒作的股票 | 短期波动大 |

## 六、机构术语

| 术语 | 解释 |
|------|------|
| **北向资金** | 通过沪港通/深港通从香港流入A股的资金（主要是外资） |
| **南向资金** | 从内地流入港股的资金 |
| **主力** | 资金量大、能影响股价走势的大资金（机构/大户） |
| **游资** | 短期进出、追求快进快出的投机性大资金 |
| **庄家** | 操控股价的大资金（违法的，但历史上存在） |
| **散户** | 资金量小的个人投资者——就是你 |

## 七、风险术语

| 术语 | 解释 | 为什么重要 |
|------|------|-----------|
| **杠杆** | 借钱来投资，放大收益也放大亏损 | 新手绝对不要用 |
| **融资** | 向券商借钱买股票 | 两融业务，需50万门槛 |
| **融券** | 向券商借股票卖出（做空） | 同上 |
| **爆仓** | 亏损超过本金，被强制平仓 | 杠杆交易的致命风险 |
| **止损** | 设定亏损上限，达到就卖出 | **最重要的风控动作** |
| **套牢** | 买入后价格下跌，卖出就亏，不想卖 | 很多散户的常态 |
| **割肉** | 在亏损状态下卖出 | 虽然痛，但有时是对的 |

## 常见误解纠正

### ❌ "涨停了还能买吗？"
不能。涨停 = 当天价格已达上限，没有人愿意在涨停价卖出，你就买不到。

### ❌ "买一手就是买一股？"
不是。一手 = 100股。买一手茅台 ≈ 18万元（按1800元/股算）。

### ❌ "T+1是说买入后第二天一定涨？"
不是。T+1是说买入后第二天才能卖出，跟涨跌无关。

### ❌ "PE越低越好？"
不一定。PE低可能是因为公司利润在下滑，市场预期不好。
