## 同花顺核心界面解读

### 主界面：分时图

```
  ┌────────────────────────────────────────────┐
  │  贵州茅台 600519  当前价: 1,798.00         │
  │  涨跌幅: +2.35%  成交量: 12,345手          │
  ├────────────────────────────────────────────┤
  │                                            │
  │   ╱╲    ╱╲╱╲                              │  ← 分时走势线
  │  ╱  ╲  ╱    ╲                             │    （黄线=均价）
  │ ╱    ╲╱      ╲                             │    （白线=实时价）
  │                                            │
  │──────────────────────────────────────────  │
  │ 9:30          11:30/13:00        15:00     │  ← 时间轴
  ├────────────────────────────────────────────┤
  │  卖5  1,800.00  100手                      │
  │  卖4  1,799.50  200手                      │  ← 五档卖盘
  │  卖3  1,799.00  500手                      │    （价格从低到高）
  │  卖2  1,798.50  300手                      │
  │  卖1  1,798.00  150手                      │  ← 最优卖价
  │──────────────────────────────────────────  │
  │  买1  1,797.50  200手                      │  ← 最优买价
  │  买2  1,797.00  400手                      │  ← 五档买盘
  │  买3  1,796.50  600手                      │    （价格从高到低）
  │  买4  1,796.00  300手                      │
  │  买5  1,795.50  200手                      │
  ├────────────────────────────────────────────┤
  │  成交: 1,798.00  1手  买入                 │
  │  成交: 1,797.50  5手  卖出                 │  ← 逐笔成交
  └────────────────────────────────────────────┘
```

**关键概念**：

### 五档盘口
- **买1-买5**：5个最高买价及对应数量 → 代表"买方愿意出多少钱"
- **卖1-卖5**：5个最低卖价及对应数量 → 代表"卖方愿意卖多少钱"
- 买1和卖1之间的差价 → **买卖价差（Spread）** → 差价小说明流动性好

> 💡 看盘口的意义：卖盘量大说明抛压重（可能要跌），买盘量大说明承接强（可能要涨）。但盘口信息可以被操纵，不要过度依赖。

### 分时线
- **白线**：实时成交价连线 → 股价的即时走势
- **黄线**：成交均价线 → 当天所有成交的平均价格
- 白线在黄线上方 → 当天多数人在更低的价格买入，目前价格偏高
- 白线在黄线下方 → 当天多数人在更高的价格买入，目前价格偏低

## 交易操作界面

### 买入操作
```
  ┌──────────────────────────────────┐
  │  股票代码: [600519          ]    │
  │  股票名称: 贵州茅台              │
  │                                  │
  │  买入价格: [1798.00        ]    │
  │  买入数量: [100            ]    │  ← 必须是100的整数倍
  │                                  │
  │  预计金额: ¥179,800              │
  │  手续费:   ¥5.1                  │
  │                                  │
  │  [确认买入]    [取消]            │
  └──────────────────────────────────┘
```

### 两种委托方式

| 方式 | 说明 | 何时使用 |
|------|------|---------|
| 限价委托 | 你指定价格，只在达到这个价格时才成交 | 想在特定价格买卖（推荐新手使用） |
| 市价委托 | 不指定价格，按当前最优价立即成交 | 急着买卖，不在乎精确价格 |

> ⚠️ **新手强烈建议只用限价委托**。市价委托可能以比你预期差的价格成交。

## 必须知道的快捷操作

### 同花顺常用功能

| 功能 | 操作 | 用途 |
|------|------|------|
| 添加自选股 | 搜索代码 → 点击"加自选" | 跟踪你关注的股票 |
| 看个股详情 | 点击自选股列表中的股票 | 查看完整信息 |
| 看板块排行 | 底部"板块"标签 | 了解哪个行业今天最活跃 |
| 看资金流向 | 个股页面 → "资金" | 看大资金在买还是卖 |
| 看龙虎榜 | "数据" → "龙虎榜" | 哪些股票今天被机构大额买卖 |

### 自选股管理
建议第一批自选股选择 5-10 只你日常能接触到的公司：
- 你喝的饮料：茅台、五粮液
- 你用的手机：苹果产业链公司
- 你穿的品牌：安踏、李宁
- 你用的软件：腾讯、阿里巴巴
- 你的学校可能涉及的公司

> 💡 从生活中能感知的公司开始，更容易理解业务。

## 交易时间总结

```
  9:15 - 9:25   集合竞价（确定开盘价）
  9:30 - 11:30   上午连续竞价（可正常交易）
  11:30 - 13:00  午间休市（不可交易）
  13:00 - 14:57  下午连续竞价（可正常交易）
  14:57 - 15:00  收盘集合竞价（确定收盘价）
```

**周末和法定节假日不交易。**
