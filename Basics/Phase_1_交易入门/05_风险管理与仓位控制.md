# 风险管理与仓位控制

> ⚠️ **这是整个Basics文件夹中最重要的一个文件。请读三遍。**

## 为什么风险管理比选股更重要？

一个残酷的事实：**90%的散户亏钱，不是因为选错了股，而是因为不会控制风险**。

| 场景 | 不风控的结果 | 有风控的结果 |
|------|-------------|-------------|
| 买入后跌10% | 不卖，等反弹 → 可能跌到50% | 止损卖出 → 只亏10%，保留本金 |
| 看到热点想追 | 全仓追入 → 一个错误就重伤 | 小仓位试 → 错了只亏小钱 |
| 连续赚了几次 | 自信心爆棚，加大仓位 → 一次大跌回到起点 | 按规则仓位 → 稳定增长 |

> **核心原则：先学会不亏钱，再学怎么赚钱。**

## 止损：你最重要的武器

### 什么是止损？

止损 = **设定一个亏损上限，达到就无条件卖出**。

就像开车系安全带——你不是"预计"会撞车，而是万一撞了你有保护。

### 止损怎么设？

| 方法 | 说明 | 适合 |
|------|------|------|
| **固定比例止损** | 跌X%就卖 | 新手最推荐 |
| **技术位止损** | 跌破支撑线就卖 | Phase 2后使用 |
| **时间止损** | X天内没涨就卖 | 短线交易用 |

**新手推荐：固定比例止损，8-10%**

### 止损的执行规则

1. **买入前就设定止损价**，不要等亏了再想
2. **触发了就无条件执行**，不要"再等等看"
3. **止损后不要立刻买回来**——你可能是在情绪驱动下操作

举例：
- 你以 50 元买入某股票
- 设定8%止损 → 止损价 = 50 × 0.92 = **46元**
- 股价跌到46元 → **立刻卖出**，不犹豫

### 为什么大多数人做不到止损？

| 心理障碍 | 真实后果 |
|----------|---------|
| "卖了就真亏了，不卖还能等回来" | 50元跌到20元，等了3年没回来 |
| "止损太频繁了，手续费都亏了" | 小亏多次 vs 一次大亏——前者远好于后者 |
| "这次不一样" | 每次都"不一样"，每次都不止损，每次都大亏 |

> 💡 **止损不是"认输"，是"保命"**。职业交易员止损频率远高于散户，但他们活得更久。

## 仓位控制：不要一把梭

### 什么是仓位？

仓位 = **你投入股市的资金占你总资金的比例**。

| 仓位 | 投入比例 | 风险 |
|------|---------|------|
| 10% | 1000元中投100元 | 极低——亏10%只亏总资金的1% |
| 30% | 1000元中投300元 | 低 |
| 50% | 1000元中投500元 | 中等 |
| 80% | 1000元中投800元 | 高——亏10%就亏总资金的8% |
| 100% | 全仓 | 极高——一次大跌就可能致命 |

### 新手仓位规则

| 规则 | 内容 | 原因 |
|------|------|------|
| **单只股票不超过总资金的20%** | 分散风险 | 一只股票跌50%，你只亏10% |
| **总仓位不超过50%** | 留一半现金 | 有余粮应对意外 |
| **第一次实盘只用5-10%资金** | 试水 | 模拟盘和实盘心态完全不同 |
| **加仓只在对的时候加** | 赚了才加 | 亏了不加——越亏越加是最致命的错误 |

### 加仓 vs 亏损加仓

| 操作 | 含义 | 结果 |
|------|------|------|
| ✅ **盈利加仓** | 股票涨了，加一点 | 赚的更多，亏损风险可控 |
| ❌ **亏损加仓（摊薄成本）** | 股票跌了，再买一点"降低平均成本" | 如果继续跌，亏的更多 |

> ⚠️ **亏损加仓是散户最致命的习惯**。你以为摊薄成本，实际上是在一个错误的决定上加大赌注。

## 资金管理：你该拿多少钱进股市？

### 大学生的资金管理建议

| 建议 | 说明 |
|------|------|
| **只用闲钱** | 亏完了也不影响生活的那部分钱 |
| **不用学费/生活费** | 绝对不借钱炒股 |
| **初始金额** | 1000-5000元足够学习（Phase 1阶段） |
| **阶段性增加** | 证明了能稳定盈利后再增加资金 |

### 为什么要用"亏完也不心疼"的钱？

因为心态是投资的核心。如果你亏了会影响生活：
- 你会焦虑 → 决策质量下降
- 你会急于翻本 → 追涨杀跌
- 你不敢止损 → 亏损越来越大

## 风险管理的完整框架

```
                    ┌──────────────────┐
                    │   总资金管理      │
                    │  (闲钱 / 分批入)  │
                    └──────────────────┘
                           │
                    ┌──────┴──────┐
                    │   仓位控制    │
                    │  (单股≤20%) │
                    └──────────────┘
                           │
              ┌────────────┴────────────┐
              │                         │
        ┌─────┴─────┐           ┌──────┴──────┐
        │   止损     │           │   分散持仓    │
        │  (8-10%)  │           │  (3-5只股票) │
        └───────────┘           └──────────────┘
```

### 简化为四条铁律

1. **只拿闲钱进股市**
2. **单只股票仓位不超过20%**
3. **买入前设止损，触发就执行**
4. **亏了不加仓，赚了才加**

## 一个反面教材

小明是大学生，拿了5000元进股市：

| 第1周 | 买了A股票，全仓5000元 | 想快速赚钱 |
|-------|----------------------|-----------|
| 第2周 | A跌了10%，亏500元 | 不止损，"再等等" |
| 第3周 | A跌了20%，亏1000元 | "摊薄成本"，再加2000元 |
| 第4周 | A跌了40%，总共亏3000元 | 心态崩溃，割肉 |
| 第5周 | 看到B股票涨了，全仓追入 | 想翻本 |
| 第6周 | B也开始跌... | 循环往复 |

**正确做法**：
| 第1周 | 买了A股票，仓位20%=1000元 | 按规则 |
|-------|----------------------|-----------|
| 第2周 | A跌8%，止损卖出，亏80元 | 只亏总资金的1.6% |
| 第3周 | 分析为什么A错了，学习 | 复盘而非加仓 |
| 第4周 | 用同样仓位尝试B | 保持纪律 |
| 结果 | 几次小亏后找到方法 | 活下来了，有机会赚 |

## 复盘：每次交易后必做

复盘 = **回顾这次交易的决策过程，找出做对了什么、做错了什么**。

### 复盘模板

| 项目 | 内容 |
|------|------|
| 买入原因 | 为什么买这只？（基本面/技术面/消息？） |
| 买入价格 | XX元 |
| 止损设定 | 跌到XX元就卖 |
| 实际止损 | 是否执行了？ |
| 卖出原因 | 为什么卖？（止损/目标到达/其他？） |
| 最终结果 | 盈/亏 XX元 |
| 做对了什么 | 哪些决策是正确的？ |
| 做错了什么 | 哪些决策是错误的？ |
| 下次改进 | 下次遇到类似情况怎么做？ |

> 💡 **坚持复盘3个月**，你会发现自己反复犯同样的错误。认清这些错误，你就进步了。

## 你现在应该做的

1. ✅ 决定你拿多少闲钱进股市（建议1000-5000元起步）
2. ✅ 写下四条铁律，贴在你能看到的地方
3. ✅ 在模拟盘练习止损——设8%止损线，严格执行
4. ✅ 模拟盘做3个月后，复盘你的每笔交易
5. ✅ 模拟盘3个月不爆仓 → 可以考虑小仓位实盘
