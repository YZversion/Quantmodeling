# K线图基础

## 一根K线告诉你什么？

K线（蜡烛图/Candlestick）是股市最基础、最重要的视觉工具。一根K线浓缩了**一个时间段内的4个价格**：

| 价格 | 含义 | 在K线上的位置 |
|------|------|-------------|
| 开盘价 | 这个时间段的第一笔成交价 | 实体的上边（跌）或下边（涨） |
| 收盘价 | 这个时间段的最后一笔成交价 | 实体的下边（跌）或上边（涨） |
| 最高价 | 这个时间段内的最高成交价 | 上影线的顶端 |
| 最低价 | 这个时间段内的最低成交价 | 下影线的底端 |

---

## K线的画法

### 涨的K线（阳线/红K）

```
        │ ← 上影线（最高价到实体顶端的距离）
    ┌───┴───┐
    │       │ ← 实体（开盘价在下，收盘价在上）
    │  红色 │    收 > 开 = 涨
    │       │
    └───┬───┘
        │ ← 下影线（实体底端到最低价的距离）
```

### 跌的K线（阴线/绿K）

```
        │ ← 上影线
    ┌───┴───┐
    │       │ ← 实体（开盘价在上，收盘价在下）
    │  绿色 │    收 < 开 = 跌
    │       │
    └───┬───┘
        │ ← 下影线
```

---

## 实体和影线的意义

### 实体
实体越长，说明开盘价和收盘价差距越大——**涨或跌的力度越强**。

| 实体 | 含义 |
|------|------|
| 很长的红实体 | 强势上涨，买方主导 |
| 很长的绿实体 | 强势下跌，卖方主导 |
| 很短的实体 | 开盘≈收盘，双方力量接近 |

### 上影线
上影线说明：**价格曾经涨到更高，但被打回来了**。

- 长上影线 → 涨上去遇到强大阻力，被打回 → 上方压力大
- 短上影线 → 涨上去没有太大阻力

### 下影线
下影线说明：**价格曾经跌到更低，但被拉回来了**。

- 长下影线 → 跌下去遇到强大支撑，被拉回 → 下方支撑强
- 短下影线 → 跌下去没有太大支撑

## 特殊K线形态

### 1. 光头光脚阳线（没有影线的红K）

```
    ┌───────┐
    │       │
    │ 红色  │ ← 开盘=最低，收盘=最高
    │       │    极强的上涨信号
    └───────┘
```

含义：从开盘就一路涨到收盘，没有任何犹豫。**极强买方力量**。

### 2. 光头光脚阴线（没有影线的绿K）

含义：从开盘一路跌到收盘。**极强卖方力量**。

### 3. 十字星（开盘≈收盘，实体极小）

```
        │
        │ ← 长上影线
    ─── ┼ ─── ← 极小实体（开盘≈收盘）
        │
        │ ← 长下影线
```

含义：多空双方激烈交战，最终旗鼓相当。**犹豫信号**——可能在转折点。

### 4. T字形（只有下影线）

含义：开盘就跌，但被强力拉回收盘价=开盘价。**下方支撑极强**。

### 5. 倒T字形（只有上影线）

含义：开盘就涨，但被强力打回。**上方压力极强**。

## K线的时间周期

一根K线可以代表不同时间段的4个价格：

| 名称 | 一根K线代表 | 用途 |
|------|-----------|------|
| 日K | 一天的开收高低 | 最常用，看趋势 |
| 周K | 一周的开收高低 | 看中期趋势 |
| 月K | 一月的开收高低 | 看长期趋势 |
| 60分钟K | 1小时的开收高低 | 看短期波动 |
| 15分钟K | 15分钟的开收高低 | 日内交易用 |

> 💡 **新手从日K开始**。周K和月K用于确认大趋势，分钟K太短容易误导。

## K线组合：最基本的3种

### 1. 红三兵（三连阳）

```
    ┌───┐
    │红 │ ← 第三根
    └───┘
    ┌───┐
    │红 │ ← 第二根
    └───┘
    ┌───┐
    │红 │ ← 第一根
    └───┘
```

含义：连续三天上涨，买方力量持续增强。**看涨信号**。

### 2. 三只乌鸦（三连阴）

含义：连续三天下跌，卖方力量持续增强。**看跌信号**。

### 3. 早晨之星（底部反转）

```
    ┌───┐
    │绿 │ ← 第一根：大跌（阴线）
    └───┘
        ┼   ← 第二根：十字星（犹豫）
    ┌───┐
    │红 │ ← 第三根：大涨（阳线）
    └───┘
```

含义：跌势中出现犹豫，然后转为涨势。**底部反转信号**。

## 成交量：K线的"证人"

K线告诉你价格发生了什么，**成交量告诉你有多少人参与了这件事**。

| 现象 | 含义 |
|------|------|
| 涨 + 量大 | 很多人参与了上涨 → 上涨可信度高 |
| 涨 + 量小 | 很少人参与了上涨 → 上涨可能是假突破 |
| 跌 + 量大 | 很多人参与了下跌 → 下跌可信度高 |
| 跌 + 量小 | 很少人参与了下跌 → 下跌可能是假恐慌 |

> 💡 **"量价配合"是技术分析的核心原则**：价格变化必须得到成交量的确认才可信。

---

## 新手常见错误

### ❌ 过度解读单根K线
一根K线的意义有限，至少要看3-5根的组合趋势。

### ❌ 把所有K线形态都当信号
K线形态只是"可能"的信号，不是"一定"的。成功率通常50-60%，不是100%。

### ❌ 只看日K不看周K
日K可能只是短期波动，周K才能看清大方向。

## 你现在应该做的

1. ✅ 在同花顺打开日K线图，识别3根阳线和3根阴线
2. ✅ 试着读出一根K线的4个价格（开、收、高、低）
3. ✅ 观察实体和影线的长度，思考它们代表什么
4. ✅ 同时看成交量，理解"量价配合"
5. ✅ 不要急着用K线做交易决策，先学会观察
