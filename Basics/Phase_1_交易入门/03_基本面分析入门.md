# 基本面分析入门

## 什么是基本面分析？

基本面分析 = **研究公司本身值多少钱**，而不是看股价走势图。

核心逻辑：**股票的长期价格终将反映公司的真实价值**。

两个问题：
1. 这家公司好不好？（质量）
2. 这家公司贵不贵？（价格 vs 价值）

---

## 基本面分析的三个层次

```
第一层：宏观（GDP、利率、政策）
    ↓ 影响所有公司
第二层：行业（行业增长、竞争格局）
    ↓ 影响行业内公司
第三层：公司（财务数据、管理层、竞争优势）
    ↓ 决定单个公司的价值
```

> 💡 **新手从第三层开始**——先学会看一家公司好不好，宏观和行业在Phase 2-3深入。

## 快速判断一家公司好不好：6个核心指标

### 1. 营收增长率 — 公司在变大吗？

| 公式 | 营收增长率 = (今年营收 - 去年营收) / 去年营收 |
|------|----------------------------------------------|

| 水平 | 含义 |
|------|------|
| > 20% | 高增长（科技公司、新能源等） |
| 10-20% | 稳健增长（消费品、制造业） |
| < 5% | 低增长或停滞 |
| 负数 | 营收在萎缩——危险信号 |

> 例子：宁德时代2023年营收增长约22%，比亚迪约42%——高增长公司。

### 2. 净利润率 — 公司赚钱效率高吗？

| 公式 | 净利润率 = 净利润 / 营收 × 100% |
|------|--------------------------------|

| 水平 | 含义 |
|------|------|
| > 30% | 极高（茅台约50%，说明商业模式极好） |
| 10-30% | 良好 |
| 5-10% | 一般（制造业常见） |
| < 5% | 很薄，一点风吹草动就可能亏损 |

### 3. ROE（净资产收益率） — 公司用股东的钱赚了多少？

| 公式 | ROE = 净利润 / 股东权益 × 100% |
|------|--------------------------------|

**这是巴菲特最看重的指标**。ROE告诉你：公司用1块钱的股东资金，能赚多少利润。

| 水平 | 含义 |
|------|------|
| > 20% | 优秀（茅台约30%，说明资本利用极高效） |
| 15-20% | 良好 |
| 10-15% | 一般 |
| < 10% | 资本利用效率低 |

> 💡 **连续5年ROE>15%的公司**，值得深入研究。

### 4. PE（市盈率） — 现在买贵不贵？

| 公式 | PE = 股价 / 每股收益（EPS） |
|------|----------------------------|

PE的本质：你花X元买1元利润。PE=20，意味着你花20元买了1元的年利润。

| 水平 | 含义（需结合行业看） |
|------|---------------------|
| < 10 | 很便宜（但可能利润在下滑） |
| 10-20 | 合理范围 |
| 20-40 | 偏贵（成长股可以接受） |
| > 40 | 很贵（除非增速极高） |
| 负数 | 公司亏损，PE无意义 |

> ⚠️ **PE不能单独看**。高PE + 高增长 = 合理；低PE + 利润下滑 = 陷阱。

### 5. PB（市净率） — 股价比公司账面价值贵多少？

| 公式 | PB = 股价 / 每股净资产 |
|------|------------------------|

| 水平 | 含义 |
|------|------|
| < 1 | 股价低于账面价值（可能被低估，也可能资产质量差） |
| 1-3 | 合理范围 |
| > 5 | 市场对无形资产（品牌/技术）估值很高 |

> 💡 PB<1的银行股很多，不代表便宜——银行有很多不良贷款风险。

### 6. 负债率 — 公司安全吗？

| 公式 | 负债率 = 总负债 / 总资产 × 100% |
|------|--------------------------------|

| 水平 | 含义 |
|------|------|
| < 30% | 很安全 |
| 30-60% | 正常 |
| > 70% | 债务沉重，有风险 |
| > 90% | 非常危险 |

> 例外：银行和地产公司负债率天生就高（商业模式决定的），不能简单用这个标准。

## 实操：用同花顺看基本面数据

### 步骤1：找到个股页面
搜索股票代码 → 打开详情页

### 步骤2：看"简况/概况"
- 公司主营业务是什么
- 所属行业
- 市值大小

### 步骤3：看"财务数据"
- 营收和净利润（最近3-5年的趋势）
- ROE、毛利率
- 负债率

### 步骤4：看"估值指标"
- PE、PB当前值和历史分布
- 和同行业对比

---

## 基本面分析的简单框架

当你研究一家公司时，回答这6个问题：

1. **这家公司做什么？** → 看主营业务
2. **它赚钱吗？** → 看净利润率和ROE
3. **它在增长吗？** → 看营收增长率
4. **它安全吗？** → 看负债率和现金流
5. **它贵不贵？** → 看PE和PB
6. **它比同行好吗？** → 看同行业对比

> 💡 如果6个问题的答案都是"是的"，这家公司可能值得关注。

---

## 基本面分析的局限

1. **滞后性**：财报是过去的数据，不代表未来
2. **不能预测短期波动**：基本面好也可能短期大跌
3. **数据可能造假**：虽然监管越来越严格，但财务造假仍存在
4. **同一数据不同解读**：PE=20有人觉得贵有人觉得便宜

> ⚠️ 基本面分析是**长期投资的锚**，不是短线交易的武器。
