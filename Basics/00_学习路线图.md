# 股市学习路线图

> **写给刚进股市的大学生：从零到专业，一步一步走。**

## 为什么要有路线图？

股市不是赌场，也不是靠运气翻盘的游戏。它是一个需要系统学习的领域——就像你学编程要先学语法、再学框架、最后做项目一样。投资也有清晰的进阶路径。

**绝大多数散户亏钱的原因**：不是不聪明，而是**跳步**——Phase 0都没走完就重仓博弈，Phase 2的内容还没看就追热点。

---

## 时间线总览

```
第1周     ─── Phase 0 完成 ─── 开户 + 模拟盘
第2-4周   ─── Phase 1 上半 ─── 交易规则 + K线 + 基本面入门
第2-3月   ─── Phase 1 完成 ─── 模拟盘3个月 + 风险管理
第3-4月   ─── Phase 2 上半 ─── 财报 + 估值
第4-6月   ─── Phase 2 完成 ─── 行业分析 + 组合管理
第6-9月   ─── Phase 3 上半 ─── 策略体系 + 宏观
第9-12月  ─── Phase 3 完成 ─── 行为金融 + 衍生品
第12月+   ─── Phase 4 启动 ─── 专业研究能力
```

## ⚠️ 三条铁律

1. **Phase 1 完成之前，只用模拟盘，不碰实盘**。模拟盘不赚钱，但实盘亏钱更快。
2. **风险管理的文件，读三遍**。大多数人的问题不是不会选股，而是不会止损。
3. **每个 Phase 有明确的完成标志**，没达到就不要跳到下一个。
