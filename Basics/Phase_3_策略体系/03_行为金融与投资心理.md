# 行为金融与投资心理

## 为什么你明明知道规则，还是亏钱？

**不是因为你不聪明，是因为你是人类。**

人类大脑在进化过程中形成的本能反应，在股市里是致命的：

| 本能反应 | 在生活中 | 在股市中 |
|----------|---------|---------|
| 追随群体 | 安全感（跟着大家走） | 追涨杀跌（跟着大家买/卖） |
| 回避损失 | 保护自己的财产 | 不止损（"卖了就真亏了"） |
| 过度自信 | 帮你勇于行动 | 盲目加仓（"这次不一样"） |
| 确认偏误 | 快速做判断 | 只看支持自己观点的证据 |

> 💡 **行为金融学研究的就是"人类为什么在投资中犯系统性错误"**。了解这些偏差，是你战胜自己的第一步。

---

## 10个最常见的认知偏差

### 1. 损失厌恶（Loss Aversion）

人们对损失的痛苦感约为同等收益快乐感的 **2-2.5倍**。

**后果**：
- 持有亏损股票太久（不愿"确认"损失）
- 过早卖出盈利股票（想"锁定"快乐）
- 结果：亏的越亏越多，赚的赚不够

**对抗方法**：
- 买入前设止损 → 把"要不要卖"变成"自动执行"
- 定期回顾：如果不持有这只股票，我今天会买吗？ → 如果不会，就该卖

### 2. 确认偏误（Confirmation Bias）

人们倾向于寻找**支持自己已有观点的信息**，忽略反对的证据。

**后果**：
- 买了某股票后，只看利好新闻，不看利空
- 忽略财报中不好的数据，只关注好的
- 在论坛里只和观点相同的人交流

**对抗方法**：
- 买入前，主动寻找3个"不应该买"的理由
- 问自己："如果我持有相反观点，我会找什么证据？"
- 定期检查：当初买入的逻辑还在吗？不在了就该调整

### 3. 过度自信（Overconfidence）

大多数人认为自己比平均水平更强——这在数学上不可能。

**后果**：
- 交易过于频繁（觉得自己每次判断都对）
- 仓位过大（觉得自己"看准了"）
- 不设止损（觉得"不会跌太多"）

**对抗方法**：
- 记录每笔交易的结果 → 数据证明你没那么准
- 限定每月最多做X笔交易 → 减少冲动操作
- 坚持8-10%止损 → 承认自己可能判断错误

### 4. 羊群效应（Herd Behavior）

人们倾向于**跟随大多数人的行为**，不做独立判断。

**后果**：
- 看到大家都在买某只股票 → 也去买（追涨）
- 看到大家都在卖 → 也跟着卖（杀跌）
- "某大佬买了XX" → 无脑跟随

**对抗方法**：
- 每次想跟风时，问自己："如果我不知道别人在买，我会买吗？"
- 独立完成自己的分析再决策
- 记住：大多数散户亏钱 → 跟着大多数 = 大概率亏钱

### 5. 近因效应（Recency Bias）

人们过于看重**最近发生的事**，忽略长期趋势。

**后果**：
- 连续赚3次 → 觉得自己"悟了"，加大仓位
- 连续亏3次 → 觉得自己"不适合投资"，放弃
- 最近市场涨 → 觉得牛市来了，全仓

**对抗方法**：
- 看至少3年的数据，不只是最近3个月
- 交易记录看整体胜率，不只是最近几笔
- 市场判断看长期估值水平，不只是最近涨跌

### 6. 锚定效应（Anchoring）

人们会被**第一个看到的价格**锚定，以此为基准判断贵贱。

**后果**：
- "这股票曾经100元，现在50元，很便宜" → 其实可能50元都贵
- "我买入价是80元，跌到60元亏很多" → 其实60元可能是合理价

**对抗方法**：
- 不看历史最高价，看当前内在价值
- 不看买入价，看当前市场价是否合理
- 重新问："如果我今天第一次看到这只股票，我会买吗？"

### 7. 处置效应（Disposition Effect）

**过早卖出盈利、过晚卖出亏损**。

这其实是损失厌恶的一种表现：
- 赚了 → 赶紧锁定 → 卖了
- 亏了 → 不想面对 → 不卖

**对抗方法**：
- 设定卖出标准（目标价或逻辑变化），不凭感觉
- 止损和止盈都按计划执行

### 8. 沉没成本（Sunk Cost Fallacy）

"我已经投入了这么多时间/金钱研究这只股票，不能放弃。"

**后果**：
- 研究了很久的公司 → 即使逻辑变了也不卖
- 已经亏了很多 → "不卖就不算亏"

**对抗方法**：
- 过去的投入是沉没成本，不影响未来收益
- 问："如果我今天才开始，我会买这只股票吗？" → 不会就该卖

### 9. 后视偏差（Hindsight Bias）

"我早就知道会涨/跌！"

**后果**：
- 回看历史觉得"一切都很明显" → 过度自信
- 觉得自己"应该能预测" → 下次更敢赌

**对抗方法**：
- 写下交易前的判断 → 事后对比 → 你没那么准
- 承认：事后看一切都很清楚，事前看一切都很难

### 10. 叙事偏误（Narrative Bias）

人们喜欢用**一个好故事**来解释复杂现象。

**后果**：
- "AI革命来了，所有AI股票都会涨" → 不看具体公司质量
- "某某CEO很厉害" → 不看财务数据
- "这个行业有大故事" → 不看估值

**对抗方法**：
- 好故事不等于好投资
- 回到数据：增速、利润率、估值——故事不能改变数字
- 问："这个故事有多少已经被定价了？"

## 自我检查清单

每次做交易决策前，问自己这5个问题：

1. ✅ 我是否因为最近赚了几次就过度自信了？
2. ✅ 我是否因为看到别人都在买就想跟风？
3. ✅ 我是否在忽略反对我观点的证据？
4. ✅ 我是否因为不想面对亏损而拒绝止损？
5. ✅ 我是否在用一个好听的故事代替数据判断？

> 如果以上任何一个答案是"是" → 延后决策24小时。
