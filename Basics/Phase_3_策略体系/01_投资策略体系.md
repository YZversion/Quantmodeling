# 投资策略体系

## 你需要找到自己的"投资身份"

Phase 1和2教了你看数据、读财报、做估值。但这些都是"工具"——**工具服务于策略，策略服务于你的目标**。

不同人有不同的投资策略，就像不同人有不同的职业选择。关键是找到适合你的那一条。

---

## 四种主要投资策略

### 1. 价值投资 — "买便宜的好公司"

| 特征 | 说明 |
|------|------|
| 核心思想 | 价格 < 内在价值时买入，等待价值回归 |
| 代表人物 | 巴菲特、格雷厄姆 |
| 关键指标 | 低PE/PB、高ROE、稳定分红 |
| 持仓周期 | 长期（3-5年+） |
| 适合人群 | 有耐心、能忍受短期波动、愿意深入研究基本面 |

**典型操作**：
- 在行业恐慌时买入龙头（如白酒行业利空时买茅台）
- 等PE回到合理水平，享受利润增长+估值回归的双重收益

### 2. 成长投资 — "买高速增长的公司"

| 特征 | 说明 |
|------|------|
| 核心思想 | 找到增速远超市场的公司，享受增长带来的股价翻倍 |
| 代表人物 | 菲利普·费雪、彼得·林奇 |
| 关键指标 | 高营收增长率、市场空间大、竞争壁垒 |
| 持仓周期 | 中长期（1-3年） |
| 适合人群 | 对新行业/新技术有洞察力、能判断增长持续性 |

**典型操作**：
- 早期识别新能源/AI行业龙头
- 接受较高PE，但要求增速支撑（PEG<1.5）

### 3. 趋势投资 — "顺势而为"

| 特征 | 说明 |
|------|------|
| 核心思想 | 不关心价值，只关心趋势——上涨时买入，下跌时卖出 |
| 代表人物 | 利弗莫尔 |
| 关键指标 | MA趋势、MACD、量价配合、突破信号 |
| 持仓周期 | 短中期（数周到数月） |
| 适合人群 | 交易经验丰富、执行力强、能严格止损 |

**典型操作**：
- 等均线多头排列后买入
- 跌破关键均线时止损（不犹豫）

### 4. 量化/系统化投资 — "用规则代替情绪"

| 特征 | 说明 |
|------|------|
| 核心思想 | 制定明确的买卖规则，机械执行，消除情绪干扰 |
| 代表人物 | 西蒙斯（Renaissance Technologies） |
| 关键指标 | 量化筛选条件、回测数据 |
| 持仓周期 | 因策略而定 |
| 适合人群 | 有编程能力、逻辑思维强、对数据敏感 |

**典型操作**：
- 设定"PE<15 + ROE>20% + 分红率>3%"筛选池
- 每月自动筛选、买入、卖出

## 大学生适合哪种策略？

| 条件 | 推荐策略 |
|------|---------|
| 时间多、资金少、能深入研究 | **价值投资**（慢慢积累） |
| 对科技/新行业有敏锐感知 | **成长投资**（但要有研究支撑） |
| 时间少、不想频繁操作 | **指数基金定投**（最省心） |
| 心态不稳定、容易追涨杀跌 | **量化筛选**（用规则控制） |

> ⚠️ **大学生不建议做趋势交易**——这需要极强的执行力和丰富的盘面经验，新手容易在频繁交易中亏损。

### 推荐起步路径

```
第1年：指数基金定投 + 价值投资学习
        → 用沪深300ETF练手，同时深入研究2-3家公司
        
第2年：尝试价值/成长投资
        → 小仓位买入深入研究过的公司
        → 开始建立自己的投资框架

第3年+：形成自己的策略体系
        → 明确"我是XX型投资者"
        → 优化和迭代策略
```

## 构建你的投资决策流程

无论哪种策略，你都需要一个**标准化的决策流程**：

```
1. 发现机会
   → 日常观察/筛选/新闻触发的关注点

2. 初步筛选（5分钟判断）
   → 行业好不好？公司龙头吗？PE合理吗？
   → 不通过 → 放入观察名单或放弃

3. 深度研究（2-3天）
   → 读最近3份财报
   → 读2-3份券商研报
   → 和同行对比
   → 估值判断

4. 制定交易计划
   → 买入价格区间
   → 仓位比例
   → 止损价格
   → 目标价格

5. 执行并跟踪
   → 按计划买入
   → 定期检查逻辑是否成立
   → 逻辑变了 → 调整或止损
```

> 💡 **90%的亏损来自跳过步骤3和4**——不研究就买入，不设止损就持有。
