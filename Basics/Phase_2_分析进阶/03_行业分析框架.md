# 行业分析框架

## 为什么行业分析比选个股更重要？

**选赛道 > 选公司**

一个衰退行业里的最好公司，可能不如一个增长行业里的普通公司。

例子：
- 2020年光伏行业爆发，普通光伏公司股价也涨3-5倍
- 同时传统地产衰退，最好的地产公司股价也腰斩

> 💡 **行业决定天花板，公司决定你能到达天花板多近的位置。**

## 行业分析的五步框架

### 第一步：行业规模与增长

| 问题 | 怎么找答案 | 关键数据 |
|------|-----------|---------|
| 行业多大？ | 行业报告、统计局数据 | 市场规模（亿元） |
| 在增长吗？ | 同上 + 公司财报中的行业数据 | 增长率（CAGR） |
| 未来3-5年会怎样？ | 政策文件、行业白皮书 | 预期增长率 |

| 增长率判断 | 含义 |
|-----------|------|
| >20% | 高速增长行业 → 机会多但竞争也激烈 |
| 10-20% | 稳健增长 → 可选好公司长期持有 |
| 5-10% | 低增长 → 看结构性机会 |
| <5%或负增长 | 衰退行业 → 避开或找转型机会 |

### 第二步：竞争格局（谁在争什么？）

#### 五种竞争格局

| 格局 | 特点 | 投资难度 |
|------|------|---------|
| **完全垄断** | 一家公司独占 | 容易（买垄断者） |
| **寡头垄断** | 2-3家主导 | 较容易（买龙头） |
| **垄断竞争** | 多家差异化竞争 | 中等（选差异化强者） |
| **完全竞争** | 大量同质化竞争者 | 困难（很难选出赢家） |
| **新兴行业** | 还没形成格局 | 高风险高回报 |

#### 用波特五力模型分析

```
        新进入者的威胁
            │
            │
 供应商议价能力 ─── 行业内部竞争 ─── 买方议价能力
            │
            │
        替代品的威胁
```

| 力量 | 强的信号 | 弱的信号 |
|------|---------|---------|
| 新进入者威胁 | 进入门槛低、技术壁垒少 | 需要大量资本/牌照/技术 |
| 供应商议价 | 供应商少、原材料稀缺 | 供应商多、替代原材料多 |
| 买方议价 | 买家少、产品同质化 | 买家分散、产品差异化大 |
| 替代品威胁 | 有更好的替代方案 | 暂无替代品 |
| 行业内部竞争 | 竞争者多、价格战激烈 | 竞争者少、各有差异化 |

> 💡 **五力都弱的行业 = 好行业**。比如白酒（进入门槛高、供应商议价弱、买家品牌忠诚、无替代品、竞争格局稳定）。

### 第三步：行业生命周期

```
     ┌───────────────────────────────────────────┐
     │                                           │
     │   萌芽期    成长期    成熟期    衰退期      │
     │                                           │
营收 │     ╱       ╱╱╱╱     ╱──────   ╲           │
增长 │    ╱      ╱      ╱           ╲╲╲          │
     │   ╱     ╱      ╱                ╲         │
     └───────────────────────────────────────────┘

利润 │     ╲       ╱╱╱╱    ╱──────    ╲           │
     │      ╲    ╱      ╱          ╲╲            │
     │       ╲  ╱      ╱             ╲           │
     └───────────────────────────────────────────┘
```

| 阶段 | 特点 | 投资策略 |
|------|------|---------|
| **萌芽期** | 市场小、不确定、大多公司亏损 | 高风险，等确定性 |
| **成长期** | 快速增长、格局开始形成 | 买增速最快的龙头 |
| **成熟期** | 增长放缓、格局稳定、利润丰厚 | 买最强龙头、享受分红 |
| **衰退期** | 市场萎缩、利润下降 | 避开，除非有转型逻辑 |

### 第四步：政策与监管环境

| 政策类型 | 例子 | 对行业的影响 |
|----------|------|-------------|
| **鼓励政策** | 新能源补贴、芯片国产化 | 行业增长加速 |
| **限制政策** | 房地产调控、教育双减 | 行业增长受限 |
| **监管收紧** | 反垄断、数据安全法规 | 行业格局可能重组 |
| **准入门槛** | 金融牌照、医药审批 | 保护已有企业 |

> ⚠️ 中国股市受政策影响极大。**不懂政策，不要投A股。**

### 第五步：行业内的关键玩家

| 排名 | 公司 | 市占率 | 核心优势 | 弱点 |
|------|------|--------|---------|------|
| 第1 | XX公司 | 30% | 品牌/规模 | 增速放缓 |
| 第2 | YY公司 | 20% | 技术/渠道 | 利润率低 |
| 第3 | ZZ公司 | 10% | 价格优势 | 品牌弱 |

> 💡 找出行业前3-5家公司，比较它们的市占率、增长率和盈利能力 → **龙头通常是最佳投资标的**。

## 行业分析的实战步骤

### 以白酒行业为例

**第一步：行业规模**
- 中国白酒市场规模约6000亿元
- 高端白酒约1500亿元
- 过去5年CAGR约10%

**第二步：竞争格局**
- 高端白酒：茅台（60%+市占率）、五粮液、泸州老窖 → **寡头垄断**
- 中低端：竞争激烈 → **完全竞争**

**第三步：生命周期**
- 高端白酒：成熟期（增长放缓但利润极高）
- 白酒整体：成熟期

**第四步：政策**
- 无明显限制政策
- 消费升级政策利好高端白酒

**第五步：关键玩家**
- 茅台：品牌无敌、ROE 30%、毛利率90%
- 五粮液：品牌第二、ROE 25%
- 泸州老窖：国窖1573增长快

**结论**：高端白酒是寡头垄断的成熟行业，政策友好，龙头盈利能力极强。**适合长期持有龙头**。

## 哪里找行业信息？

| 来源 | 内容 | 可信度 |
|------|------|--------|
| 公司年报中的"行业分析"章节 | 行业规模、竞争格局 | 高（公司自己写的） |
| 券商行业研究报告 | 深度行业分析 | 中高（可能有利益倾向） |
| 国家统计局数据 | 宏观和行业数据 | 高 |
| 行业协会报告 | 行业白皮书 | 中 |
| Wind/iFinD | 数据终端 | 高（但付费） |

> 💡 **免费渠道**：巨潮资讯网（年报）、东方财富（行业数据）、同花顺（行业对比）。
