# 投资组合管理

## 为什么不能只买一只股票？

| 场景 | 只买一只 | 分散买5只 |
|------|---------|----------|
| 某股票跌50% | 总资产亏50% | 总资产亏10%（该股占20%） |
| 某行业出问题 | 全仓受损 | 只有1-2只受影响 |
| 黑天鹅事件 | 全盘崩溃 | 部分受损，其他可能反而受益 |

> **核心原理**：分散投资不是"收益最大化"，而是"风险最小化"——确保你不会因为一个意外而全军覆没。

## 投资组合的基本原则

### 原则1：分散但不过度

| 持仓数量 | 风险分散效果 | 管理难度 |
|----------|-------------|---------|
| 1只 | 无分散 | 低 |
| 3-5只 | 基础分散 | 中 |
| 5-10只 | 良好分散 | 中高 |
| 10-20只 | 过度分散 | 高（接近指数了） |
| >20只 | 无意义 | 极高 |

> 💡 **5-10只股票是最佳区间**——足够分散但每只都能深入研究。

### 原则2：跨行业分散

**最差的组合**：5只地产股 → 地产行业出问题，全军覆没。
**最好的组合**：白酒+科技+消费+医药+金融 → 一个行业出问题，其他正常。

| 行业配置 | 目的 |
|----------|------|
| 2只消费/白酒 | 稳定收益 |
| 2只科技/新能源 | 高增长 |
| 1只金融 | 低估值+分红 |
| 1只医药 | 防御性 |
| 2只灵活仓位 | 当前机会 |

### 原则3：核心+卫星策略

```
┌───────────────────────────────────┐
│                                   │
│   核心仓位（60-70%）              │
│   → 长期持有、高质量龙头          │
│   → 茅台、腾讯、招商银行          │
│                                   │
│   ──────────────────────          │
│                                   │
│   卫星仓位（30-40%）              │
│   → 短期机会、高弹性标的          │
│   → 新能源、AI、周期股            │
│                                   │
└───────────────────────────────────┘
```

- **核心仓位**：你的"地基"，长期持有不频繁调整
- **卫星仓位**：你的"进攻"，抓住短期机会

## 仓位管理进阶

### 按确定性分配仓位

| 确定性 | 仓位比例 | 例子 |
|--------|---------|------|
| 极高确定性 | 15-20% | 茅台、长期跟踪的公司 |
| 中等确定性 | 10-15% | 行业龙头但有不确定性 |
| 低确定性 | 5-10% | 新机会、初步研究 |
| 试探仓位 | 2-5% | 刚关注、需要验证逻辑 |

> 💡 **确定性越高 → 仓位越大。不确定 → 小仓位试水。**

### 加仓规则

| 规则 | 说明 | 原因 |
|------|------|------|
| ✅ 盈利加仓 | 股票涨了5-10%后，加一部分 | 逻辑被验证，加大投入 |
| ❌ 亏损加仓 | 股票跌了后加仓摊薄 | 逻辑被否定，加大错误 |
| ✅ 分批建仓 | 不一次全买，分2-3次买入 | 降低时机风险 |
| ✅ 有止损才加仓 | 加仓前重新设止损 | 新仓位也有保护 |

## 组合的风险监控

### 定期检查清单（每周一次）

| 检查项 | 健康标准 | 不健康信号 |
|--------|---------|-----------|
| 总仓位 | <70%（留30%现金） | >90%（没有余地） |
| 单股仓位 | <20% | >30%（过度集中） |
| 行业集中度 | <30%在单一行业 | >50%（行业风险） |
| 止损触发率 | <20%的交易触发止损 | >50%（选股有问题） |
| 盈亏比 | 赚的平均大于亏的平均 | 赚的少亏的多 |

### 组合调整时机

| 时机 | 操作 | 原因 |
|------|------|------|
| 某股涨了30% | 减仓10-20% | 落袋为安，降低集中度 |
| 某股触发止损 | 执行止损 | 保护本金 |
| 新机会出现 | 用卫星仓位尝试 | 不影响核心 |
| 整体市场大跌 | 检查核心仓逻辑 | 如果逻辑不变 → 不动；变了 → 调整 |

## 不同市场环境的组合策略

| 市场环境 | 核心仓位 | 卫星仓位 | 现金比例 |
|----------|---------|---------|---------|
| **牛市** | 70%（偏成长） | 20% | 10% |
| **震荡市** | 50%（偏价值+分红） | 20% | 30% |
| **熊市** | 30%（只留最确定的） | 10% | 60% |

> ⚠️ 判断市场环境本身就是难点。新手建议**始终保持30-50%现金**，不要试图满仓博弈。

## 投资组合与指数对比

### 衡量你的组合表现

| 指标 | 你的组合 | 对比基准 |
|------|---------|---------|
| 年收益率 | X% | 沪深300收益率Y% |
| 最大回撤 | -Z% | 沪深300最大回撤 |

> 💡 **如果你的组合收益率低于沪深300** → 说明你的主动选股没有带来额外收益，不如直接买指数基金。
