# 技术指标进阶

## Phase 1回顾：你已经掌握了什么？

Phase 1学了MA（均线）、MACD、RSI三个基础指标。Phase 2要学更深入的指标，但核心原则不变：

> **指标是辅助工具，不是决策工具。多个指标一致才行动。**

---

## 指标1：KDJ（随机指标）

### 基本原理
KDJ衡量当前价格在近期价格区间中的相对位置。

- 价格接近近期最高价 → KDJ值高 → 超买
- 价格接近近期最低价 → KDJ值低 → 超卖

### 参数
- K值：快速线（敏感）
- D值：慢速线（K的平滑）
- J值：K和D的加权组合（最敏感）

### 使用方法

| 信号 | 含义 |
|------|------|
| K线从下方穿过D线 | 金叉 → 可能上涨 |
| K线从上方穿过D线 | 死叉 → 可能下跌 |
| KDJ > 80 | 超买区域 → 注意回调风险 |
| KDJ < 20 | 超卖区域 → 注意反弹机会 |
| J > 100 或 J < 0 | 极端超买/超卖 → 强烈信号 |

### ⚠️ KDJ的局限
1. **钝化**：在强势上涨/下跌中，KDJ持续停留在超买/超卖区，失去参考意义
2. **频繁信号**：在震荡市中，金叉死叉太多，容易误判
3. **适合短线**：不适合长期趋势判断

> 💡 **使用建议**：KDJ适合短线判断，配合趋势指标（MA/MACD）使用。趋势向上时，只看KDJ金叉做多；趋势向下时，只看KDJ死叉做空。

## 指标2：BOLL（布林带）

### 基本原理
布林带 = 均线 + 上下两条"弹性边界"。

- 中轨 = MA20（20日均线）
- 上轨 = 中轨 + 2倍标准差
- 下轨 = 中轨 - 2倍标准差

### 图形示意

```
         ╱ ╲          ╱╲
  上轨 ─╱   ╲  ──── ╱  ╲────
        │    ╲╱    ╲╱   │
  中轨 ─────── ──── ────────   ← MA20
        │   ╱╲    ╱╲    │
  下轨 ─╲   ╱  ──── ╲  ╱────
         ╲ ╱          ╲╱
```

### 使用方法

| 现象 | 含义 |
|------|------|
| 价格触及上轨 | 短期偏高 → 可能回调 |
| 价格触及下轨 | 立期偏低 → 可能反弹 |
| 价格在中轨附近 | 处于合理区间 |
| 布林带收窄（squeeze） | 波动率低 → 即将出现大行情 |
| 布林带张开 | 波动率高 → 行情正在剧烈波动 |

### 布林带的实战用法

**最实用的信号：布林带收窄 → 等待突破方向**

1. 布林带收窄 → 市场犹豫不决，波动率极低
2. 随后价格突破上轨或下轨 → 方向确定
3. 突破上轨 → 可能大涨；突破下轨 → 可能大跌

> ⚠️ 突破方向不确定——收窄只是告诉你"要变盘了"，不代表一定往上。

## 指标3：成交量指标进阶

### OBV（能量潮指标）

**原理**：把每天的成交量累加，涨的日子加正量，跌的日子减量。

- OBV持续上升 → 买方力量持续积累 → 看涨
- OBV持续下降 → 卖方力量持续积累 → 看跌
- OBV和价格背离 → 重要信号

**背离**：
- 价格涨但OBV跌 → 涨势没有量支撑 → 可能假涨 → 警惕下跌
- 价格跌但OBV涨 → 跌势中有人在买入 → 可能见底 → 关注反弹

### 量比

量比 = 当天成交量 ÷ 过去5天平均成交量

| 量比 | 含义 |
|------|------|
| < 0.5 | 极低量 → 市场冷清 |
| 0.5-1 | 正常量 |
| 1-2 | 放量 → 活跃 |
| 2-5 | 大量 → 异常活跃 |
| > 5 | 极大量 → 可能是转折点 |

---

## 指标4：换手率

换手率 = 当天成交量 ÷ 流通股数 × 100%

| 换手率 | 含义 | 适合的股票类型 |
|--------|------|---------------|
| < 1% | 极低 → 无人关注 | 大盘蓝筹正常 |
| 1-3% | 正常 → 日常交易 | 多数股票 |
| 3-7% | 放量 → 活跃交易 | 热门股 |
| 7-15% | 大量 → 异常活跃 | 题材股/新股 |
| > 15% | 极大量 → 疯狂交易 | 可能见顶/见底 |

> 💡 **高换手率 + 涨停 = 极热，可能见顶。高换手率 + 大跌 = 极恐慌，可能见底。**

## 多指标综合判断框架

当你看一只股票时，按这个流程：

```
1. 先看大趋势
   → MA20/MA60方向 → 价格在均线上方还是下方？
   → MACD在0轴上方还是下方？

2. 再看当前位置
   → 价格在BOLL什么位置？（上轨/中轨/下轨）
   → RSI和KDJ在什么位置？（超买/正常/超卖）

3. 再看量价关系
   → OBV方向是否和价格方向一致？
   → 换手率是否异常？

4. 综合判断
   → 多个信号一致 → 可以考虑行动
   → 信号矛盾 → 不行动，等明朗
```

## 指标的终极原则

### ❌ 不要做的事
1. 加7-8个指标，信号互相矛盾
2. 每个金叉都当买入信号
3. 逆趋势只看超买超卖做反向操作
4. 过度依赖单一指标

### ✅ 应该做的事
1. **趋势指标优先**（MA、MACD）→ 确定大方向
2. **震荡指标辅助**（KDJ、RSI）→ 找具体入场点
3. **量价确认必须**（OBV、换手率）→ 验证信号可靠性
4. **多指标一致才行动** → 3个以上指标方向一致
