# 读懂财务报表

## 三张表：公司的"体检报告"

财务报表有三张核心表，就像体检报告的不同科目：

| 报表 | 类比 | 回答的问题 |
|------|------|-----------|
| **利润表** | 收入支出表 | 这段时间赚了多少？花了多少？剩了多少？ |
| **资产负债表** | 资产负债清单 | 公司现在拥有什么？欠了什么？净资产多少？ |
| **现金流量表** | 银行流水 | 钱从哪来？花到哪去了？银行账户还剩多少？ |

> 💡 三张表互相关联——利润表的净利润会进入资产负债表的股东权益，现金流量表的净现金流验证利润的真实性。

## 利润表：公司赚了多少钱？

### 简化版利润表结构

```
营业收入                    ← 卖了多少东西
  ─ 营业成本               ← 生产这些东西花了多少
  = 毛利润                 ← 毛利 = 营收 - 成本
  ─ 销售费用               ← 广告、推广
  ─ 管理费用               ← 管理人员工资、办公费
  ─ 研发费用               ← 研发新产品投入
  ─ 财务费用               ← 借款的利息
  = 营业利润               ← 核心业务赚了多少
  + 营业外收入             ← 非核心收入（卖厂房、政府补贴）
  ─ 萦业外支出             ← 非核心支出（罚款、捐赠）
  = 利润总额               ← 交税前的总利润
  ─ 所得税                 ← 交税
  = 净利润                 ← 最终赚了多少
```

### 关键指标

| 指标 | 公式 | 意义 |
|------|------|------|
| **毛利率** | 毛利 ÷ 营收 × 100% | 产品本身的盈利能力 |
| **净利率** | 净利润 ÷ 营收 × 100% | 整体的盈利效率 |
| **费用率** | 各项费用 ÷ 营收 × 100% | 花钱效率（费用率越低越好） |

> 💡 **毛利率是商业模式好坏的晴雨表**：
- 茅台毛利率 ≈ 90%（产品成本低，品牌溢价高）
- 制造企业毛利率 ≈ 20-30%（竞争激烈，成本刚性）

### 看利润表的重点

1. **营收趋势**：连续3年是否在增长？增长率是否稳定？
2. **毛利率变化**：毛利率下降 → 可能价格竞争加剧或成本上升
3. **净利润 vs 营业利润**：如果营业利润很低但净利润很高 → 可能靠非核心收入撑着（不可持续）
4. **利润质量**：利润来自主营业务还是补贴/卖资产？

## 资产负债表：公司拥有什么、欠了什么？

### 简化版结构

```
┌──────── 资产 ─────────┐     ┌──── 负债 + 股东权益 ────┐
│                        │     │                          │
│  流动资产：            │     │  流动负债：              │
│    现金                │     │    短期借款              │
│    应收账款            │     │    应付账款              │
│    存货                │     │                          │
│                        │     │  非流动负债：            │
│  非流动资产：          │     │    长期借款              │
│    固定资产            │     │                          │
│    无形资产            │     │  东权益：               │
│    投资                │     │    股本                  │
│                        │     │    留存收益              │
│                        │     │                          │
│  ═════════════════     │  =  │  ══════════════════      │
│  总资产                │     │  总负债 + 股东权益       │
└────────────────────────┘     └──────────────────────────┘
```

**核心公式：资产 = 负债 + 股东权益**

意思是：公司拥有的一切（资产），要么是借来的（负债），要么是股东自己的（股东权益）。

### 关键指标

| 挐标 | 公式 | 意义 |
|------|------|------|
| **负债率** | 总负债 ÷ 总资产 × 100% | 借了多少钱，风险多大 |
| **流动比率** | 流动资产 ÷ 流动负债 | 短期内能还债吗？>2较安全 |
| **速动比率** | (流动资产-存货) ÷ 流动负债 | 不靠卖存货能还债吗？>1较安全 |

### 看资产负债表的重点

1. **现金够不够**：现金占流动资产的比例 → 现金多=安全
2. **应收账款**：别人欠公司的钱 → 应收账款增长快于营收=坏账风险
3. **存货**：囤了多少货 → 存货增长快于营收=可能卖不出去
4. **借款**：短期借款 vs 长期借款 → 短期借款多=偿债压力大

## 现金流量表：钱的流向

### 三种现金流

| 类型 | 含义 | 好的信号 | 坏的信号 |
|------|------|---------|---------|
| **经营现金流** | 日常做生意收到/花出的钱 | 正数且持续增长 | 负数或远低于净利润 |
| **投资现金流** | 买设备/卖资产/投资 | 合理的资本投入 | 大额不明投资 |
| **筹资现金流** | 借钱/还钱/融资/分红 | 适度融资 | 频繁借新债还旧债 |

### 为什么现金流量表最关键？

**利润可以造假，现金很难造假。**

利润表上的"净利润100亿"不一定意味着公司真的收到了100亿现金。原因：
1. 应收账款 → 利润记了，钱还没收到
2. 存货变动 → 买了原材料，利润里不直接体现
3. 折旧 → 每年减一笔账，但现金早花出去了

> 💡 **核心检验**：经营现金流是否 ≥ 净利润？
- 经营现金流 > 净利润 → 利润质量高，真收到了钱 ✅
- 经营现金流 < 净利润 → 利润可能"纸上富贵" ⚠️

### 看现金流量表的重点

1. **经营现金流是否正数且稳定增长**
2. **经营现金流是否 ≥ 净利润**
3. **自由现金流** = 经营现金流 - 资本支出 → 衡量公司真正能自由支配的钱

## 实操：在同花顺读财报

1. 打开个股页面 → 点击"财务分析"
2. 看**利润表**：近3年营收和净利润趋势
3. 看**资产负债表**：负债率、现金、应收账款
4. 看**现金流量表**：经营现金流 vs 净利润
5. 重点关注趋势变化，不只是绝对值

## 财务分析的简易框架

| 检查项 | 健康标准 | 不健康信号 |
|--------|---------|-----------|
| 营收增长 | 连续3年增长 | 增长放缓或负增长 |
| 毛利率 | >30%且稳定 | 下降趋势 |
| 净利率 | >10%且稳定 | 波动大或下降 |
| ROE | >15% | <10% |
| 负债率 | <60% | >70% |
| 经营现金流 | >净利润 | 远低于净利润 |
| 应收账款增长 | <营收增长 | >营收增长（回款困难） |
| 存货增长 | <营收增长 | >营收增长（积压） |

> 🎯 **8项都达标的公司**——极好的标的。5-6项达标——值得进一步研究。4项以下——可能有问题。
