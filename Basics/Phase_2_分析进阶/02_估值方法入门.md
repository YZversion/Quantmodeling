# 估值方法入门

## 什么是估值？

估值 = **判断一家公司"应该"值多少钱**。

核心问题：**当前股价是便宜还是贵？**

- 股价 < 内在价值 → 被低估 → 可能值得买
- 股价 > 内在价值 → 被高估 → 可能要回避
- 股价 ≈ 内在价值 → 合理定价 → 看个人判断

> ⚠️ 估值是"区间"，不是"精确数字"。你说茅台值1500-1800元，没问题；说"精确值1652.37元"，那是扯淡。

---

## 相对估值法（最常用）

### 1. PE（市盈率）— 最基础的估值指标

**PE = 股价 ÷ 每股收益（EPS）**

含义：你花多少元买1元的年利润。

| PE值 | 大致含义（需结合行业） |
|------|---------------------|
| 8-15 | 低估或价值股区域 |
| 15-25 | 合理估值区域 |
| 25-50 | 成长股溢价区域 |
| 50+ | 高估值，需极高增速支撑 |

#### PE的三种版本

| 类型 | 公式 | 特点 |
|------|------|------|
| **静态PE** | 股价 ÷ 去年EPS | 用过去数据，滞后 |
| **动态PE（TTM）** | 股价 ÷ 最近4个季度EPS | 最常用，滚动计算 |
| **前瞻PE** | 股价 ÷ 预期今年EPS | 用预测数据，有不确定性 |

> 💡 **看PE时优先看TTM PE**，最接近当前真实情况。

#### PE怎么看？

不能只看绝对值，要看**相对值**：

1. **和历史比**：当前PE处于过去5年的什么分位？
   - 低于30%分位 → 相对便宜
   - 高于70%分位 → 相对贵

2. **和同行比**：同行业其他公司PE多少？
   - 茅台PE=30，五粮液PE=25 → 茅台比五粮液贵一些
   - 但茅台ROE更高、品牌更强 → 贵一些可能合理

3. **和增速比**：PEG = PE ÷ 增长率
   - PEG < 1 → PE低于增速 → 可能被低估
   - PEG > 2 → PE远高于增速 → 可能被高估

### 2. PB（市净率）— 适合资产型公司

**PB = 股价 ÷ 每股净资产**

含义：你花多少元买1元的账面资产。

| 适用场景 | 原因 |
|----------|------|
| 银行、保险 | 资产价值明确 |
| 地产 | 土地储备有明确估值 |
| 重工制造 | 设备资产占比高 |

| 不适用场景 | 原因 |
|----------|------|
| 科技公司 | 无形资产（品牌/技术）远超账面资产 |
| 服务公司 | 资产少，赚钱靠人 |

> 💡 PB<1不一定便宜——银行PB<1很常见，因为市场担心坏账风险。

### 3. PS（市销率）— 适合亏损的成长公司

**PS = 股价 ÷ 每股营收**

含义：你花多少元买1元的年营收。

| 适用场景 | 原因 |
|----------|------|
| 还没盈利的公司 | PE无意义（亏损），但营收在增长 |
| SaaS/互联网公司 | 营收增长比利润更受关注 |

### 4. EV/EBITDA — 更专业的估值

**EV = 市值 + 总债务 - 现金**
**EBITDA = 利润 + 利息 + 税 + 折旧摊销**

含义：公司整体价值 ÷ 经营利润（不考虑资本结构差异）。

> 💡 Phase 4的专业估值会用到，Phase 2先了解概念即可。

## 绝对估值法（DCF）

DCF（Discounted Cash Flow）= **把公司未来能赚的所有现金，折算成今天的价值**。

核心思想：今天的100元和明年的100元不等价（因为有通胀和机会成本），所以明年的钱要"打折"才能和今天的钱比。

### 简化版DCF逻辑

```
未来第1年的现金流: 100亿 → 折现后 = 100 ÷ (1+10%) = 90.9亿
未来第2年的现金流: 110亿 → 折现后 = 110 ÷ (1+10%)² = 90.9亿
未来第3年的现金流: 121亿 → 折现后 = 121 ÷ (1+10%)³ = 90.9亿
...继续折现...
永续价值: 按一个增长率无限延伸 → 折现到今天

所有折现值相加 = 公司的内在价值
```

> ⚠️ DCF对假设极其敏感——增长率从5%变成4%，估值可能差30%。Phase 4会深入讲。

## 估值的实战方法

### 新手推荐：PE + 同行对比法

**步骤**：
1. 找出目标公司的TTM PE
2. 找出3-5家可比公司的PE
3. 计算同行PE中位数
4. 比较目标公司PE和同行中位数
5. 考虑目标公司的增长率和质量是否更好 → 决定PE应该高还是低

**举例**：

| 公司 | TTM PE | ROE | 增长率 |
|------|--------|-----|--------|
| 茅台 | 30 | 30% | 15% |
| 五粮液 | 25 | 25% | 12% |
| 泸州老窖 | 22 | 28% | 20% |
| **同行中位** | **25** | **27%** | **15%** |

分析：茅台PE高于中位数，但ROE也高于中位数 → 合理溢价。泸州老窖PE低于中位数但增长率高 → 可能被低估。

### 判断"贵还是便宜"的简单框架

| 判断维度 | 方法 |
|----------|------|
| 历史对比 | PE处于过去5年多少分位？ |
| 同行对比 | PE和同行业中位数比高低 |
| 增速匹配 | PEG = PE ÷ 增长率，<1偏便宜 |
| 盈利质量 | ROE是否支撑当前PE？ |

## 估值的常见误区

### ❌ "PE低=便宜"
PE低可能因为利润在下滑。一家利润持续下降的公司，PE从20变到10不是变便宜了，是基本面在恶化。

### ❌ "PE高=贵"
PE高可能因为增长快。一家增长率50%的公司，PE=50不算贵（PEG=1）。

### ❌ "不同行业用同一PE标准"
银行PE=8正常，科技公司PE=30也正常。跨行业比PE是无意义的。

### ❌ "估值能精确计算"
估值是区间，不是点。说"茅台值1500-1800"比说"值1652.37"更有意义。

## 你现在应该做的

1. ✅ 找一只股票，计算/查找其TTM PE
2. ✅ 找3-5家同行业公司，对比PE
3. ✅ 查看当前PE在过去5年的分位（同花顺可查）
4. ✅ 计算PEG（PE ÷ 增长率）
5. ✅ 综合判断：这只股票当前是偏贵还是偏便宜
