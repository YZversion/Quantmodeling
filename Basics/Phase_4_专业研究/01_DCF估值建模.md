# DCF估值建模

## DCF的本质

DCF = **Discounted Cash Flow = 现金流折现估值**

核心思想：一家公司的价值 = 它未来能产生的所有自由现金流，折现到今天的总和。

**为什么折现？**
- 今天的100元 > 明年的100元（因为你可以拿今天的100元去投资，明年变成105元）
- 所以明年的100元"折算成今天的价值"只有约95元
- 折现率就是"今天的钱比明年的钱值钱多少"的量化指标

---

## DCF的核心公式

```
公司价值 = Σ [第t年的自由现金流 ÷ (1 + WACC)^t] + 永续价值
```

三个关键输入：
1. **自由现金流（FCF）**：公司每年真正能自由支配的钱
2. **WACC（加权平均资本成本）**：折现率——公司的"资金成本"
3. **永续增长率（g）**：公司过了预测期后，每年以什么增长率永续增长

---

## 第一步：计算自由现金流（FCF）

**FCF = 经营现金流 - 资本支出**

| 项目 | 含义 |
|------|------|
| 经营现金流 | 公司做生意收到的净现金 |
| 资本支出 | 公司买设备、建厂房等投资花掉的现金 |
| FCF | 扣除必要投资后，真正"自由"的现金 |

### 为什么要用FCF而不是净利润？

- 净利润可能包含应收账款（钱没真正收到）
- 净利润扣除了折旧（但折旧不是现金支出）
- FCF是公司真正能拿出来的钱 → 更真实

---

## 第二步：估算WACC

**WACC = 权益成本 × 权益比例 + 债务成本 × 债务比例 × (1-税率)**

### 权益成本（Ke）的计算

最常用CAPM模型：
```
Ke = 无风险利率 + β × (市场预期收益率 - 无风险利率)
```

| 参数 | 中国A股常用值 | 说明 |
|------|-------------|------|
| 无风险利率 | 2.5-3.5% | 10年期国债收益率 |
| β（Beta） | 个股不同 | 股票相对于大盘的波动率 |
| 市场预期收益率 | 8-10% | 市场长期平均收益率 |

### 债务成本（Kd）
= 公司借款的利率 × (1 - 所得税率)
→ 税后债务成本，因为利息可以抵税

### 举例

假设某公司：
- 权益占比70%，债务占比30%
- Ke = 3% + 1.2 × (9% - 3%) = 10.2%
- Kd = 5% × (1 - 25%) = 3.75%
- WACC = 10.2% × 70% + 3.75% × 30% = **8.045%**

> 💡 **WACC对估值影响极大**：WACC从8%变成10%，估值可能减少30-40%。

---

## 第三步：预测未来现金流

### 两阶段模型

**阶段1：预测期（5-10年）**
- 逐年预测营收增长率 → 推算净利润 → 计算FCF
- 需要建立利润表、资产负债表、现金流量表的预测模型

**阶段2：永续期**
- 假设公司过了预测期后以一个稳定增长率永续增长
- 永续增长率通常设为2-4%（不能超过GDP长期增长率）

### 简化版DCF实操

| 年份 | FCF预测（亿元） | 折现因子(1/(1+8%)^t) | 折现值 |
|------|----------------|---------------------|--------|
| 1 | 100 | 0.926 | 92.6 |
| 2 | 110 | 0.857 | 94.3 |
| 3 | 120 | 0.794 | 95.3 |
| 4 | 130 | 0.735 | 95.6 |
| 5 | 140 | 0.681 | 95.3 |
| **预测期合计** | | | **473.1** |

### 永续价值计算

```
永续价值 = 第5年FCF × (1 + g) ÷ (WACC - g)
         = 140 × (1 + 3%) ÷ (8% - 3%)
         = 144.2 ÷ 5%
         = 2,884亿元

折现到今天 = 2,884 × 0.681 = 1,962.9亿元
```

### 总估值

```
公司总价值 = 预测期折现值 + 永续价值折现值
           = 473.1 + 1,962.9
           = 2,436亿元

每股价值 = (总价值 - 净债务) ÷ 总股本
```

> ⚠️ **永续价值通常占总估值的60-80%**——这意味着你的永续假设对估值影响最大！

## DCF的敏感性分析

DCF最大的问题：**输入参数的微小变化导致估值大幅波动**。

### WACC敏感性

| WACC | 估值（亿元） | 变化 |
|------|-------------|------|
| 6% | 3,200 | +32% |
| 8% | 2,436 | 基准 |
| 10% | 1,900 | -22% |

### 永续增长率敏感性

| g | 估值（亿元） | 变化 |
|---|-------------|------|
| 1% | 1,800 | -26% |
| 3% | 2,436 | 基准 |
| 5% | 3,600 | +48% |

> 💡 **永远做敏感性分析**——给出一个估值区间（如2,000-2,800亿元），而不是一个精确数字。

## DCF的实操建议

### 哪些公司适合DCF？
| 适合 | 原因 |
|------|------|
| 稳定增长的公司 | FCF可预测性好 |
| 成熟行业的龙头 | 永续假设合理 |

| 不适合 | 原因 |
|--------|------|
| 周期性行业 | FCF波动太大 |
| 亏损的成长公司 | FCF为负数，预测不准 |
| 金融行业 | FCF定义模糊 |

### Excel建模要点

1. 建立三表联动模型（利润表、资产负债表、现金流表）
2. 关键假设列在单独的"假设表"上 → 方便修改
3. FCF从三表推导，不是直接假设
4. 做WACC和g的二维敏感性表
