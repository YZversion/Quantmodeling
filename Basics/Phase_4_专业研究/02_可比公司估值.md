# 可比公司估值

## 什么是可比公司估值？

可比公司估值（Comps）= **找几家相似的公司，比较它们的估值倍数，推导目标公司的合理价值**。

核心逻辑：**相似的公司应该有相似的估值水平**。

> 💡 Comps是最实用的估值方法——比DCF简单，比PE单独看更严谨。

---

## Comps的四步流程

### 第一步：选择可比公司

**"可比"的标准**：

| 维度 | 要求 | 举例（对标茅台） |
|------|------|-----------------|
| 行业 | 同行业或相近行业 | 五粮液、泸州老窖、山西汾酒 |
| 规模 | 市值相近（差距不超过5倍） | 五粮液市值≈茅台的1/2 |
| 增长率 | 增速相近或可比 | 都在10-20%增长 |
| 商业模式 | 类似的盈利模式 | 都是高端白酒 |
| 地域 | 同一市场 | 都在A股 |

### 选Peer的常见错误

| ❌ 错误做法 | 原因 |
|-------------|------|
| 跨行业选Peer | 不同行业PE/PB标准完全不同 |
| 只选最便宜的Peer | 刻意压低估值区间 |
| 只选最贵的Peer | 刻意抬高估值区间 |
| 选了不可比的公司 | 比如拿银行和科技公司比PE |

### 通常选3-8家可比公司

太少 → 不够客观
太多 → 包含不可比的噪音

---

## 第二步：计算估值倍数

### 核心估值倍数

| 倍数 | 公式 | 适用场景 |
|------|------|---------|
| **PE** | 股价 ÷ EPS | 利润稳定的一般公司 |
| **PB** | 股价 ÷ BPS（每股净资产） | 银行、保险、地产 |
| **PS** | 股价 ÷ SPS（每股营收） | 还没盈利的成长公司 |
| **EV/EBITDA** | EV ÷ EBITDA | 跨行业比较、资本结构差异大 |
| **PEG** | PE ÷ 增长率 | 增长差异大的公司比较 |

### 计算Peer的倍数统计值

| 统计值 | 含义 | 用途 |
|--------|------|------|
| 平均值 | Peer的算术平均 | 可能被极端值拉偏 |
| 中位数 | Peer的中间值 | **最推荐**——不受极端值影响 |
| 25分位 | 较低的四分位 | 保守估值下限 |
| 75分位 | 较高的四分位 | 乐观估值上限 |

**举例**：白酒行业Peer的PE

| 公司 | PE |
|------|-----|
| 五粮液 | 25 |
| 泸州老窖 | 22 |
| 山西汾酒 | 35 |
| 古井贡酒 | 28 |
| **中位数** | **27** |
| **25分位** | **24** |
| **75分位** | **30** |

---

## 第三步：调整差异

**为什么不能直接用中位数？**

因为目标公司和Peer之间总有差异 → 需要调整：

| 差异维度 | 调整方向 | 举例 |
|----------|---------|------|
| 增长率更高 | PE应该更高 | 茅台增速15%，Peer平均12% → PE溢价10-20% |
| ROE更高 | PE应该更高 | 茅台ROE30%，Peer平均25% → PE溢价 |
| 规模更大 | PE可能略低（大盘股溢价低） | 茅台市值2万亿 → 规模溢价低 |
| 品牌更强 | PE应该更高 | 茅台品牌力最强 → 溢价 |

### 调整方法

**简单调整**：基于增长率差异
```
调整PE = 中位数PE × (目标增长率 ÷ Peer平均增长率)
       = 27 × (15% ÷ 12%)
       = 27 × 1.25
       = 33.75
```

**PEG调整**：
```
目标PE合理范围 = PEG合理值 × 目标增长率
               如果PEG=1.5算合理
               = 1.5 × 15% = 22.5
               如果PEG=2算合理
               = 2 × 15% = 30
→ PE合理范围约23-30
```

---

## 第四步：推导目标价

```
目标价 = 调整后的PE × 目标公司EPS

假设：
- 调整后PE范围：23-30
- 茅台2024年EPS ≈ 60元

目标价范围 = 23 × 60 ~ 30 × 60
           = 1,380 ~ 1,800元
```

### 给出估值区间而非单一值

| 情景 | PE假设 | 目标价 |
|------|--------|--------|
| 保守 | 23（25分位调整） | 1,380元 |
| 基准 | 27（中位数） | 1,620元 |
| 乐观 | 30（75分位调整） | 1,800元 |

## Comps vs DCF

| 对比 | Comps | DCF |
|------|-------|-----|
| 方法 | 相对估值 | 绝对估值 |
| 优点 | 简单、市场驱动、容易沟通 | 理论最严谨、反映内在价值 |
| 缺点 | 假设"市场定价合理"、Peer选择主观 | 假设太多、敏感性极大 |
| 适用 | 大多数情况 | 稳定增长型公司 |
| 最佳实践 | **两者结合** | DCF做理论锚，Comps做市场验证 |

> 💡 **专业分析师通常同时做DCF和Comps**：DCF给内在价值，Comps给市场定价参考。两者一致 → 更有信心；两者差异大 → 要找原因。
