# 卖方研究报告阅读指南

## 什么是卖方研究报告？

卖方 = **券商的研究部门**（如中信证券研究所、华泰证券研究所）。

他们发布的研报是：
- 对某家公司或行业的深度分析
- 包含财务预测、估值模型、评级和目标价
- **免费提供给券商客户**（散户在东方财富等平台也能看到）

> ⚠️ **"卖方"不是贬义词**——它指的是券商"卖出"服务（研究、交易通道），不是"卖出股票"。对应的"买方"是基金、保险等机构投资者，他们"买入"资产。

---

## 研报的结构：你在读什么？

一份典型的深度研报（10-30页）包含：

```
┌────────────────────────────────────────┐
│ 1. 核心观点/投资要点（1页）             │ ← 最重要！读完这个就知道要不要继续
│                                        │
│ 2. 公司概况/业务分析（3-5页）           │ ← 了解这家公司做什么
│                                        │
│ 3. 行业分析（2-3页）                   │ ← 行业趋势和竞争格局
│                                        │
│ 4. 财务分析/预测（2-4页）              │ ← 核心数据来源
│                                        │
│ 5. 估值分析（1-2页）                   │ ← DCF/Comps的具体计算
│                                        │
│ 6. 风险提示（1页）                     │ ← 必读！告诉你什么可能出错
│                                        │
│ 7. 评级和目标价（最后）                │ ← 买入/持有/卖出 + 目标价
└────────────────────────────────────────┘
```

## 读研报的正确方法

### ❌ 错误的读法

| 错误 | 原因 |
|------|------|
| 只看评级和目标价 | 评级可能受利益关系影响 |
| 直接跳到结论 | 不理解逻辑就无法判断结论是否可信 |
| 把研报当"指令" | 研报是参考，不是买卖信号 |
| 只读一份研报 | 单一来源可能有偏见 |

### ✅ 正确的读法

1. **先读"投资要点"** → 了解核心逻辑（1分钟）
2. **判断核心逻辑是否合理** → 如果不合理，后面的数据也没意义
3. **重点看财务预测** → 这是研报最有价值的部分
4. **看估值方法和假设** → 判断估值是否靠谱
5. **必读风险提示** → 了解什么可能推翻研报结论
6. **交叉验证** → 读2-3份不同券商的研报，看结论是否一致

## 研报中最有价值的部分

### 1. 财务预测表

券商分析师会给出未来3年的财务预测：

| 项目 | 2024A（实际） | 2025E（预测） | 2026E（预测） | 2027E（预测） |
|------|-------------|-------------|-------------|-------------|
| 营收 | 100亿 | 120亿 | 140亿 | 160亿 |
| 增长率 | 15% | 20% | 17% | 14% |
| 净利润 | 30亿 | 38亿 | 45亿 | 52亿 |
| EPS | 3元 | 3.8元 | 4.5元 | 5.2元 |

> 💡 **A = Actual（实际），E = Estimate（预测）**。E是分析师的判断，不是确定值。

**怎么用财务预测**：
1. 看增速假设是否合理 → 和历史增速、行业增速对比
2. 看利润率假设 → 是否在改善？改善的原因是什么？
3. 用分析师的EPS + 你自己判断的PE → 算出你的目标价

### 2. 估值部分

看券商用了什么估值方法（DCF还是Comps），关键假设是什么。

| 要检查的 | 为什么 |
|----------|--------|
| WACC假设 | 太低 → 估值偏高 |
| 永续增长率 | 太高 → 估值偏高 |
| 可比公司选择 | 是否真的可比？ |
| 目标PE | 是否和增长率匹配？ |

### 3. 风险提示

这是很多人忽略但极其重要的部分：

常见风险提示类型：
| 类型 | 含义 | 你应该做什么 |
|------|------|-------------|
| 行业竞争加剧 | 利润率可能下降 | 持续跟踪毛利率变化 |
| 政策风险 | 监管可能收紧 | 关注政策动向 |
| 宏观风险 | 经济下行影响需求 | 判断宏观环境 |
| 估值风险 | 当前PE偏高 | 设更保守的目标价 |

> 💡 **风险提示不是废话**——如果研报提到的风险真的发生了，结论就需要调整。

## 研报的评级体系

| 评级 | 含义 | 券商常用表述 |
|------|------|-------------|
| 买入 | 预期涨幅 > 20% | "强烈推荐"/"买入" |
| 增持 | 预期涨幅 10-20% | "推荐"/"增持" |
| 中性 | 预期涨幅 -10~10% | "持有"/"中性" |
| 减持 | 预期跌幅 > 10% | "卖出"/"减持" |

> ⚠️ **券商很少给出"卖出"评级**——因为卖出评级可能得罪上市公司（影响未来业务），也可能得罪持有该股的客户。所以"中性"可能隐含了"其实不建议买"。

---

## 研报的局限性

### 1. 利益冲突
券商的研究部门和投行部门有业务关系 → 可能倾向于给出正面评级。

### 2. 羊群效应
大多数券商的结论方向一致 → 如果你只看一致意见，可能错过变异认知。

### 3. 时效性
研报发布后，市场可能已经反应了 → 跟着研报操作可能晚了。

### 4. 过度乐观
分析师的盈利预测普遍偏高 → 实际结果往往低于预测。
