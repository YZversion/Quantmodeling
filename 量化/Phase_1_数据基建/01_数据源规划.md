## 一、数据需求总览

按你的8个研究领域，需要三类数据：

| 数据类型 | 用途 | 频率 | 必要性 |
|----------|------|------|--------|
| **行情数据** | 日线/周线/月线 OHLCV + 指数 | 日更 | 必须 |
| **财报数据** | 季报/年报的营收、利润、资产负债 | 季更 | 必须 |
| **宏观数据** | GDP/CPI/利率/PMI/资本开支 | 月更 | 重要 |
| **供应链数据** | 出货量/订单/产能利用率 | 月更/季更 | 你的alpha |
| **事件数据** | 政策发布/产品发布/财报日期 | 事件驱动 | 辅助 |
| **情绪数据** | 新闻/社交媒体NLP分析 | 日更 | 你懂LLM，加分项 |

## 二、免费数据源详解

### 2.1 Tushare（推荐首选）

| 项目 | 说明 |
|------|------|
| 网址 | https://tushare.pro |
| 安装 | `pip install tushare` |
| 注册 | 需要注册获取token |
| 免费版 | 200积分（注册即送），日线/财报/指数可获取 |
| 高级版 | 5000积分/年（约500元），分钟线/资金流/龙虎榜 |

**免费版能获取的关键数据**：

```python
import tushare as ts
pro = ts.pro_api('你的token')

# 日线行情
df = pro.daily(ts_code='000001.SZ', start_date='20200101', end_date='20260630')

# 季度财报
df = pro.income(ts_code='000001.SZ', period='20251')

# 每日指标（PE/PB/换手率等）
df = pro.daily_basic(ts_code='000001.SZ', start_date='20200101')

# 指数行情
df = pro.index_daily(ts_code='000300.SH', start_date='20200101')
```

**高级版额外数据**：
| 数据 | 说明 | 量化价值 |
|------|------|---------|
| 分钟线 | 1min/5min/15min/30min/60min | 日内策略 |
| 资金流 | 个股主力/散户资金进出 | 情绪因子 |
| 龙虎榜 | 每日涨停股买卖明细 | 游资动向 |
| 融资融券 | 每日两融余额 | 杠杆情绪 |

### 2.2 AKShare（备用+补充）

| 项目 | 说明 |
|------|------|
| 网址 | https://akshare.xyz |
| 安装 | `pip install akshare` |
| 特点 | 完全免费，覆盖面广，但稳定性略低于Tushare |

**AKShare独有的有用数据**：
```python
import akshare as ak

# 港股数据（联想集团等）
df = ak.stock_hk_daily(symbol="00992")  # 联想集团

# 美股数据（NVIDIA等）
df = ak.stock_us_daily(symbol="NVDA")

# 宏观经济数据
df = ak.macro_china_gdp()  # GDP
df = ak.macro_china_cpi()  # CPI

# 北向资金
df = ak.stock_em_hsgt_north_net_flow_in_em()

# 行业板块行情
df = ak.stock_board_industry_name_em()
```

### 2.3 东方财富（研报+资金流）

| 项目 | 说明 |
|------|------|
| 获取方式 | Web页面手动/爬虫 |
| 价值数据 | 研报全文、个股资金流、机构持仓 |

### 2.4 yfinance（美股/港股补充）

```python
import yfinance as yf

# NVIDIA日线
nvda = yf.Ticker("NVDA")
df = nvda.history(period="5y")

# 联想集团港股
lenovo = yf.Ticker("0992.HK")
df = lenovo.history(period="5y")
```

## 三、数据存储架构

### SQLite + 文件混合方案

```
量化/
├── data/
│   ├── db/
│   │   ├── stocks.db         ← SQLite：日线/财报/因子
│   │   ├── macro.db          ← SQLite：宏观数据
│   │   └── supply_chain.db   ← SQLite：供应链映射+传导数据
│   ├── csv/
│   │   ├── sector_map.csv    ← 板块映射表
│   │   ├── factor_dict.csv   ← 因子字典
│   │   └── events.csv        ← 事件日历
│   └── raw/
│   │   ├── tushare_cache/    ← Tushare原始数据缓存
│   │   └── reports/          ← 研报PDF/HTML
│   └── chroma/
│   │   └── news_embeddings/  ← ChromaDB：新闻语义向量
```

### 数据更新pipeline

```
[每日 18:00]
    │
    ├── Tushare → 拉取当日行情 → 写入 stocks.db
    ├── AKShare → 拉取北向资金 → 写入 stocks.db
    ├── 东方财富 → 爬取资金流 → 写入 stocks.db
    │
    └── [每周一次]
    │   └── Tushare → 拉取周度财报更新 → 写入 stocks.db
    │   └── AKShare → 拉取宏观数据 → 写入 macro.db
    │
    └── [每月一次]
    │   ├── 因子重算 → 写入 stocks.db factor表
    │   ├── 供应链数据更新 → 写入 supply_chain.db
    │   └── ChromaDB新闻embedding更新
```

> 💡 你已经会Docker了——可以把这个pipeline做成Docker容器定时运行。
